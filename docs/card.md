# Считывание и обработка карт

Данное SDK поддерживает работу с такими типами карт как:
* Ultralight 48Byte
* Ultralight Ev1 48Byte
* Ultralight Ev1 128Byte
* Ultralight C 144Byte
* Classic ID 64Byte
* Classic 1Kb
* Classic 4Kb
* Plus 4Kb
* Plus SE 1Kb
* Desfire Ev1 2Kb
* Desfire Ev1 4Kb
* Desfire Ev1 8Kb
* MIR (Карты МИР)
* Банковские карты
* Приложение PERCo.Доступ
* EM-Marine и HID
